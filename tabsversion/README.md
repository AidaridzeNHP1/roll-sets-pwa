# Наборы роллов PWA

Это веб-версия каталога, которую можно открыть на iPhone в Safari и добавить на экран Домой.

## Как запустить на телефоне

1. Загрузите содержимое этой папки на HTTPS-хостинг: GitHub Pages, Netlify, Vercel или любой обычный сайт.
2. Откройте ссылку на iPhone в Safari.
3. Нажмите кнопку «Поделиться».
4. Выберите «На экран Домой».
5. Откройте приложение с иконки.

Для PWA важен HTTPS. Если просто отправить `index.html` файлом в Telegram или открыть из «Файлов», офлайн-кэш и установка как приложение обычно не заработают.

## Где править данные

- `sets.json` — наборы и роллы внутри набора.
- `recipes.json` — составы роллов.
